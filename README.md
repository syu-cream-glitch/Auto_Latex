# 📊 LaTeX表作成ツール

NotionやExcelからコピーした表をLaTeX形式に変換したり、インタラクティブに表を作成してLaTeXコードを生成するWebツールです。

## ✨ 機能

### 📋 Notion表の貼り付け
- NotionやExcelから表をコピーして直接貼り付け
- タブ区切りで自動解析
- ヘッダー行の自動検出
- 空列の自動除去

### 🎨 インタラクティブ表作成
- 行数・列数を自由に設定（最大20行×10列）
- **列名のカスタマイズ**: 各列に任意の名前を設定可能
- リアルタイム編集: 表のセルを直接編集
- 動的行追加: 行数を自由に増減

### ⚙️ LaTeX設定
- **キャプション**: 表のタイトル設定
- **ラベル**: 参照用のラベル設定
- **位置**: 表の配置位置（here/top/bottom/page）
- **キャプション位置**: キャプションの上下を設定

### 💾 エクスポート機能
- **CSV**: 表データをCSV形式でダウンロード
- **HTML**: 表をHTML形式でダウンロード
- **LaTeX**: 生成されたLaTeXコードをファイルとしてダウンロード

## 🚀 インストール

### 必要条件
- Python 3.12+
- uv (パッケージマネージャー)

### セットアップ
```bash
# リポジトリをクローン
git clone <repository-url>
cd gw-2025-b3-2-3

# 依存関係をインストール
uv sync
```

## 🎯 使用方法

### 起動
```bash
# Streamlitアプリを起動
uv run streamlit run main.py
```

ブラウザで `http://localhost:8501` にアクセスしてください。

### 📋 Notion表の貼り付け
1. NotionやExcelで表を選択してコピー（Ctrl+C）
2. 「📋 Notion貼り付け」タブを選択
3. テキストエリアに貼り付け（Ctrl+V）
4. 必要に応じてヘッダー設定を調整
5. LaTeX設定を行い、コードをコピー

### 🎨 インタラクティブ作成
1. 「🎨 インタラクティブ表作成」タブを選択
2. **表のサイズを設定**: 行数と列数を指定
3. **列名を設定**: 各列に任意の名前を入力
4. **新しい表を作成**: ボタンをクリックして表を生成
5. **表を編集**: 各セルをクリックして値を入力
6. **LaTeX設定**: キャプション、ラベル、位置を設定
7. **コードをコピー**: 生成されたLaTeXコードを使用


## 📖 LaTeXでの使用例

生成されたLaTeXコードを文書に組み込む例:

```latex
\\documentclass{article}
\\usepackage[utf8]{inputenc}

\\begin{document}

\\begin{table}[h]
    \\centering
    \\caption{サンプル表}
    \\label{tab:sample}
    \\begin{tabular}{lcc}
        \\hline
         & 項目A & 項目B \\\\
        \\hline
        データ1 & 100 & 200 \\\\
        データ2 & 150 & 250 \\\\
        \\hline
    \\end{tabular}
\\end{table}

\\end{document}
```

## ⚠️ 注意事項

- **特殊文字**: セル内の `&`、`%`、`$` などのLaTeX特殊文字は自動エスケープされません
- **複雑な表**: セル結合や複雑な書式には対応していません
- **データ型**: 数値や文字列の自動判別は行いません
- **ブラウザ**: 最新のChrome/Firefox/Safariを推奨
