# 📊 LaTeX表作成ツール

Web上で表を作成し、リアルタイムでLaTeX形式のコードを生成するシンプルなツールです。

## 機能

- **📏 自由な表サイズ**: 行数と列数を自由に設定（最大20行×10列）
- **📝 リアルタイム編集**: 表のセルを直接編集可能
- **📄 LaTeXコード生成**: 編集内容を即座にLaTeX表形式に変換
- **📋 コピー機能**: 生成されたLaTeXコードを簡単にコピー
- **👀 プレビュー**: オプションで表のプレビュー表示

## インストール

1. 依存関係をインストール:
   ```bash
   uv sync
   ```

## 実行

1. Streamlitアプリを起動:
   ```bash
   uv run streamlit run main.py
   ```

2. ブラウザでアクセス（通常 http://localhost:8501）。

## 使用方法

1. **表のサイズを設定**: 行数と列数を指定
2. **「新しい表を作成」をクリック**: 指定サイズの空の表が作成されます
3. **表を編集**: 各セルをクリックして値を入力
4. **LaTeXコードを確認**: 表の下にリアルタイムでLaTeX形式のコードが表示されます
5. **コードをコピー**: 下のテキストエリアからLaTeXコードをコピー

## LaTeXでの使用例

生成されたLaTeXコードを文書に組み込む例:

```latex
\\documentclass{article}
\\begin{document}

\\begin{tabular}{|c|c|c|}
\\hline
列1 & 列2 & 列3 \\\\
\\hline
A & B & C \\\\
\\hline
1 & 2 & 3 \\\\
\\hline
\\end{tabular}

\\end{document}
```

## 特徴

- **シンプルなUI**: 直感的に操作可能
- **リアルタイム更新**: 表の変更が即座にLaTeXコードに反映
- **コピーしやすさ**: コードブロックとテキストエリアの両方でコードを提供
- **柔軟な編集**: Streamlitのdata_editorを使用した快適な編集体験

## 注意事項

- セル内の特殊文字（&、%、$など）はLaTeXで適切にエスケープする必要があります
- 複雑な表形式（結合セルなど）には対応していません
- 生成されたコードは基本的なtabular環境を使用しています
